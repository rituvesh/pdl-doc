APIer og hendelser

Integrasjon mot Persondataløsningen er enkelt og fleksibelt! Velg selv hvilke data du ønsker å motta og hvordan du ønsker å motta dem.

Oppslag

GraphQL-inspirert REST-endepunkt, sikret og GDPR-compliant. Slå opp hvilke som helst personidenter og få ønskede opplysningstyper tilbake.

Søk

Vi tilbyr fleksible søkemuligheter med lynraske svar, sortert etter treffsikkerhet. Lag egne søk eller benytt Lucene Query Language.

Hendelser

Vi leverer med stolthet til "Livet er en strøm av hendelser"! Motta hendelser i sanntid om endringer i persondata. Hendelsene er rike og inneholder hele opplysningstypen.

Tidsriktig og presist

Nå mottar vi opplysninger fra Folkeregisteret og øvrige kilder umiddelbart endringene er registrert hos dem. Det betyr at NAV ikke lengre oppdateres med gårsdagens data. Alle endringer i grunnleggende personopplysninger strømmes videre i sanntid til alle applikasjoner som ønsker å bruke livshendelser i sin arkitektur.

Arkitektur for fremtiden

Persondataløsningen er samfunnskritisk og tilbyr de mest brukte dataene i NAV. Dette stiller helt egne krav til arkitekturen vår, samtidig som vi lager en løsning som skal leve lenge. For å imøtekomme dette bygger vi applikasjonene små og løskoblede med Java og Spring Boot, Apache Kafka og event sourcing. Alle våre applikasjoner kjører på NAVs Kubernetes-baserte plattform Nais!
